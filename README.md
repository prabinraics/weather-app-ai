# Weather App (PHP + JavaScript)

## Description
This project is a dynamic weather application that fetches real-time weather data using an API and displays it based on user input. The app allows users to search for any city and view current weather information through a clean and responsive interface.

## Features
- Search weather by city name
- Real-time weather data using API
- Dynamic and responsive user interface
- Local storage caching for faster access
- Error handling for invalid city input

## Technologies Used
- HTML (Structure)
- CSS (Styling & Layout)
- JavaScript (Frontend logic & API calls)
- PHP (Backend API handling)
- OpenWeather API

## How It Works
- User enters a city name
- JavaScript sends request to backend/API
- PHP processes API data
- Weather information is displayed on UI
- Data may be stored in local storage

## How to Run
1. Start a local server (XAMPP / WAMP / Live Server)
2. Place project folder inside server directory
3. Open browser and run:
   http://localhost/your-project-folder
4. Enter a city name and view weather details

## Project Structure
- index.html (UI structure)
- style.css (design & styling)
- script.js (frontend logic)
- weather.php (API handling)
- README.md (documentation)
